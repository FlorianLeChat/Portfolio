#informations

À propos de moi

Je suis Florian, un développeur français de 22 ans. Je suis principalement un développeur back-end (JS & NodeJS, PHP, ...) mais je suis vraiment intéressé par l'apprentissage du front-end et du design UI/UX.

J'ai également de l'expérience dans les langages de script et de programmation comme le Lua, le Python et le C# (avec .NET Core). Cela inclut évidemment connaissances solides dans les systèmes de sauvegarde SQLite et MySQL utilisés également dans le Web.

J'apprends à programmer depuis l'âge de 14 ans.

#projects

Projets

Raven Framework

Le framework Raven (sources fermées) est un ensemble d'outils pour accélérer la création de scripts simples, mais également de systèmes d'interaction homme-machine plus complexes. Bâti au dessus de l'API Lua (GLua) du jeu vidéo Garry's Mod, le framework a subit plusieurs itérations pour améliorer la robustesse de ses mécanismes et comportements. Il est actuellement utilisé par la communauté Combined-Roleplay, mais également disponible (en partie) sur le Workshop de Steam.

Facepunch Commits Monitor

Le moniteur des commits Facepunch (sources ouvertes) est l'une de mes toutes premières applications graphiques écrites en C# et propulsées grâce à .NET 5 et Windows Forms. Grâce à une interface graphique simple et l'API du studio de développement Facepunch, il est possible de suivre en temps réel les changements des dépôts publics Git du studio et de les afficher sous forme de notifications Windows.

Florian's Bot

Le robot de Florian (sources fermées) est un simple robot utilisant les services Discord pour s'exécuter. Utilisant NodeJS et le paquet NPM DiscordJS, le robot est capable d'effectuer des actions de modération simples de la part des propriétaires des serveurs. Il est également relié à une base de données MySQL pour permettre la sauvegarde et la récupération des informations utilisateurs.

Simple File Storage

Le système de stockage simplifié de système (sources ouvertes) est mon tout premier projet utilisant les langages du web les plus répandus (HTML, CSS et PHP) afin de permettre aux utilisateurs d'enregistrer des fichiers quelconques sur un serveur. Actuellement, il est possible de mettre en ligne des fichiers, mais également de contrôler le contenu sauvegardé grâce à une page d'administration sécurisé par mot de passe. Pour l'instant, les fondations sont fragiles, mais à l'avenir, le système possédera un système de compte utilisateurs et une page d'accueil pour que les utilisateurs/administrateurs puissent accéder aux contenus sauvegardés facilement.

Python RPG

Python RPG (sources ouvertes) est ma première application console écrite en Python. Il avait pour but d'expérimenter ce langage et d'exploiter la simplicité de sa syntaxe pour réaliser un projet dans le thème du roleplay. Le programme est actuellement capable de lire des histoires permettant à l'utilisateur d'interagir avec un univers fictif (ou non) via différents moyens, comme des actions envers un objet ou un personne, mais aussi des dialogues entre l'utilisateur et des personnages non-joueurs. Pour l'instant, le programme fonctionne seulement en mode de lecture, mais il est prévu que j'intègre un mode de création et d'édition des histoires.
